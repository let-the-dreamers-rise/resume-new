# Ashwin Goyal - Terminal Portfolio

A terminal-style minimalist portfolio website for Ashwin Goyal, built using Next.js, React, Tailwind CSS, and Framer Motion.

## Features

- **Terminal Interface**: Command-line styled layout mimicking a Unix shell.
- **Interactive File Viewers**: Navigate through experience, projects, skills, and achievements via simulated terminal files.
- **Bilingual Support**: Toggle between English and Chinese languages.
- **Responsive Design**: Clean and interactive layout across mobile and desktop displays.

## Getting Started

First, install dependencies with Bun:

```bash
bun install
```

Then, run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
